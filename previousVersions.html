Lab 4 Changes

Hello friends. If you're new to this website this tab will seem kinda pointless, but if you've been keeping up with all the iterations then it will let you know what exactly changed this time around. The first new feature is this section. I already told you what it does so we won't go over it again. Some other changes were the addition of CSS so that the site can stop looking like it was made in the 90s. The CSS gave us a spiffy new menu at the top that is way more radical than some lame bullet list. It also added a rad effect whenever you mouse over the menu options. Go ahead and try it out, you will probably only be slightly disappointed by how lackluster it is after all this hype. You might also be thinking that these nifty little description paragraphs look different, and you would be right! For starters all the headers are now the same size, and the body of the paragraph is now in white text! This was done because white text is generally easier to read on most backgrounds, and it makes the bodies of the paragraphs stand out more from the titles. The menu at the top also had it's text color changed to white so that it would stick out more. You might also notice a neat little picture to the left of this that says “innovation”. That picture was added because it looks cutting edge and I thought the site could use some new age ideas. Anyway, that's all that's new for now. Be on the look out for more bodacious changes in the future!
